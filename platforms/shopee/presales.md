# Shopee 售前客服话术

> 专业的 Shopee 售前客服话术模板，涵盖砍价应对、优惠券使用、多站点物流、产品咨询等东南亚电商场景。

## 使用说明

- 话术中的 `[变量]` 需要根据实际情况替换
- Shopee 覆盖东南亚多个站点，注意根据站点调整物流和支付说明
- 建议配合快捷回复软件使用，提升响应速度

---

## 1. 砍价应对

**适用场景：** 客户询问能否降价、要求优惠

```
您好！感谢您的询价。这款商品的价格已经是我们的最低价了，因为我们直接从工厂采购，价格非常有竞争力。不过，如果您现在下单，我可以为您申请[赠品/优惠券]。您觉得怎么样？
```

---

## 2. 多站点物流说明

**适用场景：** 客户询问配送方式、物流时效

```
您好！关于物流，Shopee在不同站点有不同的物流方式。对于[站点名称]，我们使用[物流方式]，预计[时间]内送达。物流费用已包含在商品价格中，您无需额外支付。如有疑问，随时咨询我。
```

---

## 3. 优惠券使用

**适用场景：** 客户询问优惠券如何使用

```
您好！关于优惠券，Shopee平台经常会有各种优惠券活动。您可以在Shopee首页或活动页面领取优惠券。使用优惠券时，请确保满足使用条件（如最低消费金额）。如果您有优惠券问题，我可以帮您查看。
```

---

## 4. 商品咨询

**适用场景：** 客户询问商品详情

```
您好！关于[商品名称]，我可以为您详细介绍。这款商品的特点是[特点1]、[特点2]、[特点3]。商品详情页面有详细的图片和说明，您可以查看。如果您还有其他问题，随时咨询我。
```

---

## 5. 商品推荐

**适用场景：** 向客户推荐相关产品

```
您好！根据您对[商品A]的兴趣，我推荐您看看[商品B]。这款商品具有[特点1]、[特点2]等特点，很多购买[商品A]的客户也购买了[商品B]，反馈都很好。您想了解一下吗？
```

---

## 6. 库存紧张提醒

**适用场景：** 热销商品库存不足时

```
您好！感谢您对这款商品的关注。目前库存比较紧张，只剩[数量]件了。这是热销商品，经常断货。如果您感兴趣，建议尽快下单，以免缺货。如有疑问，随时咨询我！
```

## 7. 商品对比说明

**适用场景：** 客户询问不同商品的区别

```
您好！关于[商品A]和[商品B]的区别，主要在于：[区别1]、[区别2]、[区别3]。根据您的需求[需求描述]，我建议您选择[推荐商品]，因为[推荐理由]。您觉得怎么样？
```

**适用场景：** 客户对购买安全有疑虑


---

## 使用技巧

1. **砍价应对**：不要直接拒绝，提供替代方案（赠品、优惠券）
2. **多站点注意**：Shopee 覆盖多个东南亚国家，物流和支付方式不同
3. **活动配合**：结合 Shopee 平台活动（如 9.9、11.11）提升转化
4. **语言适配**：根据站点使用当地语言（中文、英文、泰语等）
