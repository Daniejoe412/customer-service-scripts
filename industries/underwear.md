# 内衣客服话术

> 专业的内衣行业客服话术模板，涵盖尺码测量、面料舒适度、款式推荐、退换货处理等全场景。

## 使用说明

- 话术中的 `[变量]` 需要根据实际情况替换
- 内衣属于贴身衣物，客户对尺码和面料要求高
- 注意隐私保护，话术要专业得体

---

## 售前话术（10条）


## 使用技巧

1. **尺码推荐要专业**：内衣尺码复杂，要耐心询问
2. **隐私保护**：强调隐私包装，让客户放心
3. **退换货说明清楚**：贴身衣物退换有要求，提前说明
4. **话术得体**：内衣话术要专业，避免尴尬
