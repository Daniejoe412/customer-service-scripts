# 童装客服话术

> 专业的童装行业客服话术模板，涵盖尺码年龄对照、面料安全、洗涤保养等全场景。

## 使用说明

- 话术中的 `[变量]` 需要根据实际情况替换
- 童装客户最关心面料安全和尺码准确性
- 建议配合快捷回复软件使用，提升响应速度

---

## 售前话术（10条）

### 1. 产品核心卖点
**适用场景：** 客户问"这件衣服怎么样"

```
亲，这款童装很受欢迎～
✨ 亮点：
• 面料：[A类/纯棉]，亲肤透气，宝宝穿着舒服
• 设计：[设计特点]，活泼可爱
• 做工：无骨缝制，不磨皮肤
宝宝多大了？我帮您推荐尺码～
```

## 使用技巧

1. **强调安全性**：童装客户最关心面料安全，主动说明A类标准
2. **尺码要留余量**：小朋友长得快，建议买大一点
3. **洗涤提醒**：建议新衣服先洗再穿，体现专业
4. **售后要重视**：涉及宝宝的问题要快速响应
