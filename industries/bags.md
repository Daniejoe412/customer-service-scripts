# 箱包客服话术

> 专业的箱包行业客服话术模板，涵盖材质说明、容量尺寸、真假辨别、退换货等全场景。

## 使用说明

- 话术中的 `[变量]` 需要根据实际情况替换
- 箱包客户关注材质、容量和耐用性
- 建议配合快捷回复软件使用

---

## 售前话术（10条）

### 1. 产品核心卖点
**适用场景：** 客户问"这款包怎么样"

```
亲，这款很受欢迎～
✨ 亮点：
• 材质：[真皮/PU/帆布]，[质感/耐用]
• 容量：可装[物品描述]
• 设计：[设计特点]
需要了解具体尺寸吗？
```

---

### 2. 尺寸容量咨询
**适用场景：** 客户问"能装多少/多大"

```

• [物品1，如13寸电脑]
• [物品2，如A4文件]
• [物品3]
```

## 使用技巧

1. **尺寸要具体**：说清能装什么，客户更直观
2. **材质要说明**：真皮/PU区别要说清
3. **保养要指导**：不同材质保养方法不同
4. **五金要保证**：五金质量是客户关注点
